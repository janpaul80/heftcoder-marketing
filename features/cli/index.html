Powerful Commands

heft init [name]

Scaffold a new HeftCoder project with your choice of templates. Includes React, TypeScript, Tailwind, and all dependencies pre-configured.

heft generate "prompt"

Generate components, pages, or features using natural language. The AI understands your project structure and creates matching code.

heft dev

Start a local development server with hot reload. See your changes instantly as you build.

heft deploy

Deploy your application to HeftCoder's global CDN with a single command. Custom domains and SSL included.

heft pull / heft push

Sync your local project with HeftCoder cloud. Work offline and push changes when ready.

heft templates

Browse and install from our library of production-ready templates. Dashboards, landing pages, e-commerce, and more.
